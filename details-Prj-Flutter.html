Projet réalisé

Description du Projet : Développement complet d'une application e-commerce utilisant le framework Flutter et une base de données Firebase. L'application offre une expérience utilisateur fluide pour la recherche de produits, la gestion du panier d'achat, et le processus de paiement. Les fonctionnalités clés incluent l'authentification utilisateur, l'intégration Firebase Cloud Firestore pour le stockage des données, et l'utilisation de Firebase Storage pour héberger les images des produits.
Responsabilités et Réalisations :
Analyse des Besoins : Identification des exigences fonctionnelles et non fonctionnelles de l'application. Définition des fonctionnalités clés telles que la gestion des produits, le panier d'achat, et l'authentification utilisateur. Configuration et Intégration Firebase : Création et configuration d'un projet Firebase pour l'authentification, la base de données et le stockage. Intégration des SDK Firebase dans le projet Flutter. Conception de l'Interface Utilisateur (UI) : Conception de wireframes et de maquettes pour assurer une expérience utilisateur intuitive. Utilisation de Flutter pour créer une interface réactive et esthétique.
  • Mise en Place de la Structure de Projet : Organisation du projet en modules pour une gestion efficace. Utilisation du package cloud_firestore pour faciliter l'interaction avec Firebase Firestore.
  • Gestion des Produits et Panier d'Achat : Création d'une collection "produits" dans Firestore. Mise en œuvre du panier d'achat avec mise à jour en temps réel.
  • Authentification Utilisateur : Mise en place de l'authentification utilisateur avec Firebase Authentication. Stockage sécurisé des données utilisateur dans Firestore.
  • Processus de Paiement : Intégration des fonctions Firebase pour gérer les transactions de paiement. Utilisation de Firestore pour enregistrer et suivre les commandes.
  • Notifications et Fonctionnalités Avancées : Mise en œuvre de notifications Cloud Messaging pour informer les utilisateurs des offres spéciales. Utilisation du stockage Firebase pour héberger les images des produits.
  • Tests et Optimisation : Réalisation de tests unitaires et d'intégration pour garantir la stabilité de l'application. Optimisation des performances pour assurer une expérience utilisateur fluide.
  • Déploiement et Maintenance : Déploiement de l'application sur les plateformes Google Play Store et Apple App Store. Maintenance continue, correction des bugs et mises à jour régulières.
  • Technologies Utilisées : Flutter Dart Firebase (Authentication, Firestore, Cloud Functions, Cloud Storage)
